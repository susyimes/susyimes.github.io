## 记录摘要

把接口能力开放出来，让 Agent 可以根据任务构建模型智能小程序。

## 复盘点

- 问题：很多任务需要临时 UI 和临时工具，但不值得人工完整开发。
- 方案：开放接口能力，让 Agent 生成小程序、连接模型、执行任务。
- 边界：生成的小程序需要权限声明、数据边界和可撤销机制。
