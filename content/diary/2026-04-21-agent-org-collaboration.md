## 记录摘要

把 Agent 组织从单线执行扩展到协作、对抗、分布和并行，让不同 Agent 承担不同判断角色。

## 复盘点

- 协作：不同 Agent 共享目标和上下文。
- 对抗：审查 Agent 挑错，避免执行 Agent 自说自话。
- 分布/并行：把探索、实现、验证拆开，提高速度和可靠性。
