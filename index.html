01

Context as fresco

上下文不是备注，而是壁画底稿。每次任务、失败路径、工具调用和复盘都成为下一次行动的视觉层。

02

Harness as architecture

自主性来自循环控制、权限边界、风险阈值和完成判定，不只是模型能力。

03

Skill as craft

把重复行动炼成可复用技能，把个人经验变成 Agent 可以接住的操作结构。

04

Android as sensorium

手机侧状态、无障碍树、使用统计和任务中心，让 Agent 从真实设备状态里生成 purpose。

Narrative Rail

一条从人到 Agent 的长卷

不是按项目堆卡片，而是沿着“意图如何被机器接住”展开：观察、组织、执行、复盘、再进入下一轮。

Act I

Human gives direction

人类给方向、价值、边界和判断标准，但不提前封死模型的探索路径。

Act II

Context keeps memory

日志摘要、证据指针、失败原因和下一步被压缩为可长期复用的上下文。

Act III

Harness governs action

权限、trace、循环、工具调用和人类确认点决定 Agent 能否持续、安全地行动。

Act IV

Agents form an organization

Work、Observe、Assist 三类 Agent 从多开窗口，变成真正的协作组织。

Act V

Review becomes ritual

复盘不是收尾文档，而是下一次 Agent 自进化的燃料。
