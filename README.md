# susyimes.github.io

Personal GitHub Pages site for `susyimes`.

This site migrates the public content from `susyimes/agent-workflow-portfolio` and reframes it as a living archive for AI Agent application engineering, SVMES flow notes, Android-side agent work, skills, harness design, and human-agent handoff practice.

## Content migrated

- `data/diary.json`: 32 diary records from the original portfolio.
- `data/agentdiary.json`: preserved as an additional diary source.
- `data/svmes-flow.md`: full SVMES flow archive.
- `assets/screenshots/`: preserved for future sanitized evidence images.
